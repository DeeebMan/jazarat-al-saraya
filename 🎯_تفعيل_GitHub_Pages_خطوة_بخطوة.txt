🌐 تفعيل GitHub Pages خطوة بخطوة

═══════════════════════════════════════════════════════════════════════════════

📋 الخطوة الأولى: الوصول للمستودع
═══════════════════════════════════════════════════════════════════════════════

1. اذهب إلى: https://github.com/DeeebMan/jazarat-al-saraya
2. ستجد نفسك في الصفحة الرئيسية للمستودع

═══════════════════════════════════════════════════════════════════════════════

🔧 الخطوة الثانية: الوصول للإعدادات
═══════════════════════════════════════════════════════════════════════════════

1. في أعلى الصفحة، ستجد عدة تبويبات:
   • Code
   • Issues
   • Pull requests
   • Actions
   • Projects
   • Wiki
   • Security
   • Insights
   • Settings ← اضغط على هذا

2. اضغط على تبويب "Settings"

═══════════════════════════════════════════════════════════════════════════════

📄 الخطوة الثالثة: البحث عن Pages
═══════════════════════════════════════════════════════════════════════════════

1. في القائمة الجانبية اليسرى، ستجد:
   • General
   • Access
   • Security
   • Code and automation
   • Integrations
   • Pages ← ابحث عن هذا

2. إذا لم تجد "Pages" في القائمة، ابحث في شريط البحث عن "pages"

═══════════════════════════════════════════════════════════════════════════════

⚙️ الخطوة الرابعة: إعداد Pages
═══════════════════════════════════════════════════════════════════════════════

1. في صفحة Pages، ستجد قسم "Source"
2. اختر "Deploy from a branch"
3. في قسم "Branch"، اختر "main"
4. اضغط "Save"

═══════════════════════════════════════════════════════════════════════════════

⏳ الخطوة الخامسة: انتظار التفعيل
═══════════════════════════════════════════════════════════════════════════════

1. بعد الضغط على Save، انتظر بضع دقائق
2. ستظهر رسالة "Your site is published at..."
3. ستحصل على رابط مثل:
   https://deeebman.github.io/jazarat-al-saraya/

• جميع الوظائف متاحة ومفعلة
