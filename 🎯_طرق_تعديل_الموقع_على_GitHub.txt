✅ الطريقة الأولى: التعديل المحلي (الأفضل)
═══════════════════════════════════════════════════════════════════════════════

📝 الخطوات:
1. عدّل الملفات على جهازك (index.html، admin.js، styles.css، إلخ)
2. انقر مرتين على ملف "UPDATE_WEBSITE.bat"
3. انتظر حتى يتم رفع التحديثات
4. سيظهر التحديث على الموقع خلال بضع دقائق

💡 المميزات:
• سهولة التعديل
• إمكانية تعديل عدة ملفات مرة واحدة
• يمكنك اختبار التغييرات محلياً قبل الرفع

═══════════════════════════════════════════════════════════════════════════════

🌐 الطريقة الثانية: التعديل مباشرة على GitHub
═══════════════════════════════════════════════════════════════════════════════

📝 الخطوات:
1. اذهب إلى: https://github.com/DeeebMan/jazarat-al-saraya
2. اضغط على الملف الذي تريد تعديله
3. اضغط على أيقونة القلم (✏️) للتعديل
4. عدّل المحتوى
5. اكتب وصف للتغيير في "Commit message"
6. اضغط "Commit changes"

💡 المميزات:
• سريع للتعديلات الصغيرة
• لا يحتاج برامج إضافية
• التحديث فوري

═══════════════════════════════════════════════════════════════════════════════

📱 الطريقة الثالثة: التعديل عبر لوحة الإدارة
═══════════════════════════════════════════════════════════════════════════════

📝 الخطوات:
1. اذهب إلى رابط الموقع/login.html
2. سجل دخولك بالبيانات:
   • اسم المستخدم: admin
   • كلمة المرور: sarayadeeb
3. استخدم لوحة الإدارة لإضافة/تعديل/حذف المنتجات
4. جميع التغييرات تُحفظ في localStorage

💡 المميزات:
• سهل الاستخدام
• لا يحتاج معرفة تقنية
• واجهة مستخدم جميلة

🔄 أنواع التعديلات الممكنة
═══════════════════════════════════════════════════════════════════════════════

🎨 تعديل التصميم:
• تغيير الألوان في ملف styles.css
• تعديل الخطوط والأحجام
• إضافة/إزالة عناصر

📦 تعديل المنتجات:
• إضافة منتجات جديدة
• تعديل الأسعار والأوصاف
• إضافة/تغيير الصور

🔧 تعديل الوظائف:
• تعديل ملفات JavaScript
• إضافة ميزات جديدة
• تحسين الأداء

📄 تعديل المحتوى:
• تغيير النصوص
• تحديث معلومات الاتصال
• إضافة أقسام جديدة
